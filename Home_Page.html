User Profile and Account Management


Available on the top-right section of your dashboard or home page, is the Manage Account section. From here the user logged into the system can view or edit their profile, access the API tokens, or Logout of the billing platform.


We will have a look at the options available with each:


User Profile


The first available option presented on the list when clicking on your credentials is; Profile.

There are five editable options available to update your personal preferences or information.


The following options describes the processes related to each:

Option 1: The user can update their account's profile by adding a new photo, and changing their account user name and/or e-mail address.

image to add


Option 2: The user can change their current password with a new password. To ensure the password stays secure, it is recommended to use at least 8 digits.

image to add


Option 3: Enable two factor authentications. When this feature is enabled, the user will receive a random secure token during authentication which can be retrieved from their phone's Google Authenticator application.

image to add


Option 4: Logout of other browser sessions, allows the user to logout of all their browser sessions across all their devices. In the case that the user suspect that their account was compromised, it is advisable that they also change their current password.

image to add


Option 5: : A user can delete their account from the system if this is required. It should be noted that once an account was deleted, all the resources and data associated with the account will also be deleted.
